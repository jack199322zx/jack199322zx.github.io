你是否也想搭建个人博客？这里教大家一个免费搭建个人博客的方法。

写在开篇之前

当你想要搭建个人博客的网站，却为负担昂贵的服务器费用而苦恼，那么本文为您提供了一个解决方案。

let’s begin

前提准备：搭建好 node 和 git环境， 准备一个github账号。
关于环境搭建可以参考下边的一篇博文，里面已经讲解的比较详细。
https://blog.csdn.net/ainuser/article/details/77609180

then

登录github账号，创建一个仓库。

这里需要注意的是，仓库的名称必须是username.github.io

创建的过程中可能会有些漫长，需要等待一段时间方可访问。

进入刚创建的仓库，打开Settings，找到github pages

可以配置自定义域名映射，也就是可以域名解析到阿里云域名等。
这里要注意个人网站的代码只能提交到master主分支，且不可修改。否则是无法展示你提交的页面的。

github ssh配置

github账号配置完成之后，需要配置ssh，方便hexo的部署。
本地通过 ssh-keygen -t rsa -C 邮箱地址
创建密匙，最后得到了两个文件：id_rsa（私匙）和id_rsa.pub（公匙）
打开id_rsa.pub，将其中的内容复制到github账号的ssh keys设置中。
输入ssh -T git@github.com，测试添加ssh是否成功。
可以参考另一篇博文https://www.cnblogs.com/fengxiongZz/p/7707219.html

last

修改_config.yml
在deploy下设置repository: ssh://…（你的仓库地址）
执行hexo clean, hexo generate, hexo deploy。（以后只要改动了代码，都可以通过这种方式发布）
最后访问你的github pages，如zhangsan.github.io，看到最新的页面即发布成功。
